# Diagrams - Army Challenge

Comprehensive collection of MermaidJS diagrams that visualize the architecture and system flows for the military unit simulation.

---

## How to Use the Diagrams

### Visualization

All diagrams are in **MermaidJS** format. They can be viewed:

1. **Directly on GitHub**: Automatically rendered
2. **In Markdown Preview**: Using extensions like Markdown Preview Mermaid Support
3. **Online**: At [MermaidJS Playground](https://mermaid.ai/play#pako:eNqrVkrOT0lVslJSqgUAFW4DVg) by copying the content
